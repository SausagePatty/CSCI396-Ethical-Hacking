### Running the web crawler program

## Requirements
- Python 3.x
- To run the script, modify the `meta_ip` variable in the script to match the IP address of your Metasploitable VM.
- Install `requests` library (via `pip install requests`) if it's not already installed using the command.

## Usage
1. **On Ubuntu Machine**:
	- Ensure `dirs.txt` is in the same directory as `webCrawler.py`.
	- Open terminal and run the script using Python 3:

 	```bash
	python3 webCrawler.py
	```
	- Upon executation, found directory will be printed out on the terminal.
